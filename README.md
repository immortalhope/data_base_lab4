# data_base_lab4
Лабораторна робота 4 

У даній лабораторный роботі виконується робота з MongoDB(pymongo)

Для коректної роботи даної лр, необхідно завантажити файл lab4.py та confіg.py, а також До цієї ж папки додати файли з даними: "Odata2019File.csv" та "Odata2020File.csv".
У файлі confіg.pу ввести необхідну конфігурцію бд, після чого можна запускати lab4.py.
